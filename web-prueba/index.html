¿Qué es asegurar un equipo?

Asegurar un equipo informático significa protegerlo frente a ataques, virus, pérdida de datos o accesos no autorizados. Igual que cerramos la puerta de casa, debemos cerrar las puertas digitales de nuestros dispositivos

Como mantener el equipo actualizado

Transcripción:

1. Mantén el sistema operativo siempre actualizado.

2. Activa las actualizaciones automáticas del antivirus.

3. No ignores las notificaciones de seguridad.
